## Setup

### Requirements
- python 3.11+

### Dev Setup (One-Time)
1. In root, create venv

    ```bash
    python -m venv .venv
    ```

2. Activate .venv

    ```bash
    # For Mac:
    source .venv/bin/activate

    # For Windows:
    .venv\Scripts\Activate
    ```

3. Install requirements

    ```bash
    pip install -r requirements.txt
    ```
